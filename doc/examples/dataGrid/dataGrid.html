This is an example of FLAP in action.


Download Flash 6 player from here if you don't have it. It even works on Linux!

This is an example of a GUI control (Data Grid) that would be non-trivial to implement in straight HTML. Note that by clicking on column headers the values are sorted by that column, and the columns can be resized. Click on a grid row to open the browser!
The datagrid component is not freely available, unfortunately. You can buy it here from Macromedia.

This is the server-side Perl script dataGrid.pl.
